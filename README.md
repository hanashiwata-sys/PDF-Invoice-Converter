# PDF Invoice Converter

PDF納品書をExcelへ変換するツールです。

## 機能

- PDF一括読み込み
- 会社名抽出
- 日付抽出
- 商品明細抽出
- Excel出力

## 使用ライブラリ

- pdfplumber
- pandas
- openpyxl

## 実行方法

'''bash python pdf_invoice_to_excel.py

## サンプルデータ

sample_pdfフォルダにサンプルPDFを格納しています。

変換結果はsample_outputフォルダに格納しています。
